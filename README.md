Отличная идея. Систематизация стека поможет вам и вашей команде при дальнейшей поддержке или масштабировании проекта.
Ниже представлена полная документация по техническому стеку и архитектуре нашего лендинга, структурированная для удобного импорта в ваш Notebook или базу знаний (Notion, Obsidian, Confluence).

Технический стек: Усачев Недвижимость (Лендинг De Luxe)

2. Фреймворки и Библиотеки (Frontend)
Tailwind CSS (via CDN)
Роль: Утилитарный CSS-фреймворк для быстрой и консистентной верстки.
Конфигурация: Кастомная тема с фирменными цветами (black: '#030303', coal: '#0b0b0d', platinum: '#e5e7eb', gold: '#C1A063').
Особенности: Поддержка режима darkMode: 'class', использование SRI (Subresource Integrity) для безопасной загрузки.
AOS (Animate On Scroll) v2.3.1
Роль: Библиотека для плавного появления элементов при скролле.
Настройки: once: true (анимация проигрывается один раз), easing: 'ease-out-quart', смещение (offset) 50px.
Google Fonts
Шрифт: Montserrat (начертания 200, 300, 400, 500, 600).
Оптимизация: Использование preconnect к доменам Google для ускорения загрузки (First Contentful Paint).
3. Пользовательский интерфейс и паттерны (UI/UX)
Color Theme Management: Реализован скрипт переключения светлой/темной темы с запоминанием выбора пользователя в localStorage браузера и проверкой системных настроек (prefers-color-scheme).
Glassmorphism: Эффект матового стекла (backdrop-blur, полупрозрачные заливки) для изоляции текстовых блоков от динамичного видеофона.
Responsive Grid: Адаптивная сетка на основе Flexbox и CSS Grid (Tailwind), перестраивающаяся от мобильного вида (1 колонка) до десктопа (4-5 колонок).
4. Безопасность и защита данных (Security)
Модуль
Описание
CSRF Protection
Клиентская генерация токена через crypto.getRandomValues() с сохранением в document.cookie (атрибуты Strict и Secure) и проверкой при отправке формы.
Rate Limiting
Локальная защита от спама в формах (не более 3 попыток отправки за 1 минуту), реализованная через JS-объект formRateLimiter.
SRI (Subresource Integrity)
Проверка хеш-сумм внешних скриптов (Tailwind, AOS) для защиты от подмены кода на стороне CDN.
